# 📌 4) UX / Interaction Design (Step-by-Step)

## 🏁 A) Starting Interaction

**Screen structure:**

```
+--------------------------------------+
| ToolBar (left) | Canvas (center)     |
|                |                     |
| Units / Snap   | Visualization pane  |
| Price Panel    |                     |
+--------------------------------------+
```

### 🖱 Drawing Flow (Walls & Cabinets)

**1) User selects tool:**

* Wall
* Base Cabinet
* Wall Cabinet
* Tall Cabinet

**2) User clicks on canvas** →
→ **Start point set**
→ Immediately shows **Floating Dimension Input**

**3) Dimension Input Behavior**

* Cursor is on start point
* Floating UI: `[___ cm / m]`
* User types number immediately
* Press Enter → finalizes shape (length)
* During typing, canvas shows **preview rectangle** with depth depending on type

**4) Preview + Snap Indicators**

* While moving mouse after first click:

  * If cursor near snap target (Corner/Endpoint):
    → Show **highlighted dot**
    → Show **guideline helper**
  * If not near snap target:
    → No snap highlight

**5) Press Enter** → finalize unit
**6) Press Esc** → cancel drawing

## 🧲 C) Magnetic Snapping Rules

### 🔹 Snap Targets

Only these:

1. **Wall Corners (vertices)**
2. **Wall Endpoints (edges)**
3. **Cabinet Endpoints**

**No Snap to midpoint by default**

### 🔹 Priority Order

1. Corner
2. Endpoint
3. (Optional if toggled) Midpoint
4. Grid (only if enabled)

### 🔹 Snap Radius

* Default: ~8–16 px (adjust based on zoom)
* If cursor inside radius → snap activates
* Show visual feedback

---

## 🔀 D) Edge/Corners-Only Snap Behavior

When dragging or drawing:

```
if distanceToEdge < snapRadius:
   snapPoint = edge
   showEdgeHighlight()
else:
   noSnap
```

**Important:**

* No snapping to grid or arbitrary points unless toggle is ON
* Snapping only from edges/corners increases design consistency

---

## 🧱 E) Wall Connection & Corner Creation

When drawing a new wall near an existing wall:

1. If the end point is within snap radius of a wall endpoint → snap
2. The new wall endpoint merges with existing wall’s vertex
3. A **shared corner vertex** is created
4. If angle is approximately 90°, auto normalize
5. If angle > tolerance → show angle helper

**Rule:**

```
if distance( newWallEnd, existingWallEnd ) <= snapRadius:
   newWallEnd = existingWallEnd
   unifyVertex()
```

# 📌 5) Cabinet Auto Split Logic

### Example:

* Base Cabinet: 3.5m
* Tall Cabinet placed at 2m
  System does:
* Split base into: 2m + 1.5m
* Tall is placed exactly in middle

Algorithm:

```
remainingLength = base.length - tallOffset
Base1 = tallOffset
Base2 = remainingLength
```

Refactor geometric endpoints accordingly
